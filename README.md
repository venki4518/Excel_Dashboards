# 📊 Excel Dashboards

This folder contains multiple dashboards built in **Microsoft Excel**, showcasing skills in data analysis, visualization, and automation. Each dashboard includes datasets, KPIs, and interactive visuals designed to support business decision‑making.

---

## 📂 Files Included
- **Chocolate_sales_dashboard.xlsx**  
  - Focus: Chocolate sales performance  
  - KPIs: Total sales, regional performance, top products  
  - Features: Pivot tables, charts, slicers, seasonal demand analysis  

- **covid19__Realmartsales_dashboard.xlsx**  
  - Contains two dashboards in one file:  
    1. **COVID‑19 Dashboard** → Tracks cases, recoveries, fatalities, and regional spread  
    2. **Real Mart Sales Dashboard** → Analyzes revenue, customer footfall, and product performance  
  - Features: Pivot tables, charts, slicers, automated reporting workflows  

---

## 📊 Dashboard Highlights
- **Chocolate Sales Dashboard:** Identifies top‑selling categories, regional demand, and seasonal trends.  
- **COVID‑19 Dashboard:** Provides visibility into pandemic progression and recovery rates.  
- **Real Mart Sales Dashboard:** Supports retail decision‑making with insights into revenue growth and efficiency gains.
